      설치없이 인터넷 웹페이지 저장방법( html, mht, pdf )

      • leverage
      • Info / IT
      • 2016. 3. 24.

      뉴스 글이나 커뮤니티 사이트 글 같은 경우는 즐겨찾기를 해놔도 삭제가 되면

      다시 볼 수 없게 될 때가 있습니다. 이 때 웹페이지 통째로 저장을 하면 되는데

      3가지 방법을 소개하겠습니다. html, mht, pdf 순으로 소개합니다.


      html로 저장하기

      구글 크롬에서 저장할 페이지를 띄운 후

      Ctrl + S 를 누르면 다른 이름으로 저장 창이 뜹니다.

      파일 형식을 웹페이지_완료 선택 한후 원하는 위치에 저장하면 완료


      mht로 저장하기

      인터넷 익스플로러에선 Ctrl + S 를 누르면

      웹 페이지 저장 창이 뜹니다. 파일 형식은 웹페이지 보관파일(*.mht) 를 선택한 후 저장합니다.


      빨간색 네모 안에 있는 파일 2개는 html 형식으로 저장한 파일이며

      흰색 네모 안에 있는 파일 1개는 mht 형식으로 저장한 파일입니다.

      html 인 경우 파일 2개가 세트이므로 둘 중 하나라도 없으면 사용 못합니다.


      pdf로 저장하기

      구글 크롬에서 Ctrl + P 를 눌러 인쇄페이지 창을 띄웁니다.

      좌측 메뉴에서 변경을 눌러 PDF로 저장을 클릭합니다.


      좌측 메뉴에서 옵션 - 머리글과 바닥글 체크 해제 해주세요.

      그 다음 저장 클릭


      이렇게 바로 .pdf 로 저장 가능합니다.
